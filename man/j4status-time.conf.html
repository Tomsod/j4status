j4status-time.conf(5)

j4status time plugin configuration

Name

j4status-time.conf — j4status time plugin configuration

Synopsis

Configuration for the time plugin

The time plugin use the main j4status configuration file (see j4status.conf(5)).

Description

It controls the time plugin behavior.

Sections

Section [Time]

Format= (format string, defaults to "%F %T")

Format string used to display date and time.

See strftime(3).

Zones= (list of timezone names, defaults to local)

Timezones to print the time for.

See strftime(3).

Formats= (list of format string, defaults to empty)

Each timezone may have its own format string. You must have exactly the same number of format strings as timezones for the Zones key.

An empty format string tells the plugin to use the Format key format.

See strftime(3).
