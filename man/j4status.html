j4status(1)

status line generator

HomeMan pages

Name

j4status — status line generator

Synopsis

j4status [-o plugin] [-i plugin...]

Description

j4status is a status line generator to track remote or local events and do actions the user wants to.

j4status is composed by a core, the application itself, and various plugins. Plugins may be output plugins or input plugins.

Official plugins

Output plugins

The following output plugins are distributed with j4status:

flat

which displays flat text

i3bar (see j4status-i3bar.conf(5))

which implements the i3bar JSON protocol

Input plugins

The following input plugins are distributed with j4status:

time (see j4status-time.conf(5))

a plugin to display date and time

file-monitor (see j4status-file-monitor.conf(5))

a plugin to monitor files and dispaly their content

nm (see j4status-nm.conf(5))

a NetworkManager client plugin, to display network status

upower

a UPower client plugin, to display battery status

sensors (see j4status-sensors.conf(5))

a sensors watching plugin, to display temperatures frow your motherboard sensors

systemd (see j4status-systemd.conf(5))

a systemd unit monitoring plugin, which displays the status of systemd units

Options

-o, --output=plugin

Specify the output plugin

-i, --input=plugin

Specify an input plugin

May be specified multiple times.

-1, --one-shot

Tells j4status to stop right after starting

-V, --version

Print version

-h, --help

Show help options

Environment

$J4STATUS_PLUGINS_DIR

Additional directory to browse for plugins.

This directory is browsed first, before standard directories.

$J4STATUS_CONFIG_FILE

Additional configuration file to check.

This file is checked first, before standard files, to override them.
