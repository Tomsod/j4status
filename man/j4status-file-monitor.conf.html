Name

j4status-file-monitor.conf — j4status file-monitor plugin configuration

Synopsis

Configuration for the file-monitor plugin

The file-monitor plugin use the main j4status configuration file (see j4status.conf(5)).

Description

It controls the file-monitor plugin behavior.

Sections

Section [FileMonitor]

Files= (list of file names or paths)

Files to monitor in the plugin runtime directory.

You can specify absolute paths too.

Directory

$XDG_RUNTIME_DIR/j4status/file-monitor

The directory in which the plugin will search files.

If $XDG_RUNTIME_DIR is not set, we use $XDG_CACHE_HOME, with a fallback to ~/.cache.
