Name

j4status-nm.conf — j4status NetworkManager plugin configuration

Synopsis

Configuration for the NetworkManager plugin

The NetworkManager plugin use the main j4status configuration file (see j4status.conf(5)).

Description

It controls the NetworkManager plugin behavior.

Sections

Section [NetworkManager]

Interfaces= (list of interfaces names, defaults to empty)

The list of interfaces the plugin will monitor.

ShowUnknown= (boolean, defaults to false)

Whether to show or not interfaces with unknown state.

ShowUnmanaged= (boolean, defaults to false)

Whether to show or not interfaces with unmanaged state.

HideUnavailable= (boolean, defaults to false)

Whether to hide or not interfaces with unvailable state.
