j4status-systemd.conf(5)

j4status systemd plugin configuration

Name

j4status-systemd.conf — j4status systemd plugin configuration

Synopsis

Configuration for the systemd plugin

The systemd plugin use the main j4status configuration file (see j4status.conf(5)).

Description

It controls the systemd plugin behavior.

Sections

Section [systemd]

Units= (list of unit names, defaults to empty)

The list of units the plugin will monitor.
