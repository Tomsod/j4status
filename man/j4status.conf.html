j4status.conf(5)

j4status configuration file

Name

j4status.conf — j4status configuration file

Synopsis

The config files are searched in following order (first found with the relevant section is used):

  1. $J4STATUS_CONFIG_FILE (see j4status(1))

  2. $XDG_CONFIG_HOME/j4status/config (fallback to ~/.config/j4status/config)

  3. /etc/j4status/config

  4. /usr/share/j4status/config

Description

This is the configuration file for j4status(1). The syntax of this file is described in the XDG Desktop Entry Specification.

This file settings can be overridden by passing arguments to j4status invocation directly.

Plugins may parse their own section(s) in this file for their configuration.

Sections

Section [Plugins]

Output= (plugin name, defaults to flat)

Output plugin to use.

Input= (list of plugin names)

List of input plugins to use.
