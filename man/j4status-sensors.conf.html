j4status-sensors.conf(5)

j4status sensors plugin configuration

Name

j4status-sensors.conf — j4status sensors plugin configuration

Synopsis

Configuration for the sensors plugin

The sensors plugin use the main j4status configuration file (see j4status.conf(5)).

Description

It controls the sensors plugin behavior.

Sections

Section [Sensors]

Sensors= (list of sensor names, defaults to empty)

The list of sensors the plugin will watch.

They may contain wildcards. The default empty value makes the plugin watch all the available sensors.

See the output of sensors(1).
