Name

j4status-i3bar.conf — j4status i3bar plugin configuration

Synopsis

Configuration for the i3bar plugin

The i3bar plugin use the main j4status configuration file (see j4status.conf(5)).

Description

It controls the i3bar plugin behavior.

Sections

Section [i3bar]

NoStateColour= (hexadecimal #-prefixed colour or none, defaults to none)

Colour to use for sections without state.

UnavailableColour= (hexadecimal #-prefixed colour or none, defaults to #0000FF)

Colour to use for sections with unavailable state.

BadColour= (hexadecimal #-prefixed colour or none, defaults to #FF0000)

Colour to use for sections with bad state.

AverageColour= (hexadecimal #-prefixed colour or none, defaults to #FFFF00)

Colour to use for sections with average state.

GoodColour= (hexadecimal #-prefixed colour or none, defaults to #00FF00)

Colour to use for sections with good state.
