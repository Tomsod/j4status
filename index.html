j4status

A plugin-based status line generator

j4status provides a status line using several plugin to retrieve information from your computer.

j4status is a part of the j4tools tools set.

Dependencies

  • GLib
  • For the i3bar output plugin:
    • yajl (2.0.0 or later)
  • For the NetworkManager input plugin:
    • libnm-util (0.9 or later)
    • libnm-glib (0.9 or later)
  • For the Upower input plugin:
    • libupower-glib
  • For the sensors input plugin:
    • libsensors from lm_sensor
  • For the systemd input plugin:
    • gdbus-codegen utility from GLib
